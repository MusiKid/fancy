## Usage
You can use the CLI or the GUI (WIP).

Set fans speeds:

```sh
fancy set -f 0 5.5
```

Set config:

```sh
fancy set -c "CONFIG"
```

Set auto:

```sh
fancy set -a
```

Get fans speeds:

```sh
fancy get speeds
```

Get temperatures:

```sh
fancy get temps
```

Get current config:

```sh
fancy get config
```

Get recommeneded configs:

```sh
fancy list --recommended
```

Get all available configs:

```sh
fancy list
```
